# Census Income Prediction (Data Analysis & Machine Learning)

## 📌 Problem Statement

The objective of this project is to predict whether an individual's income exceeds $50K per year based on demographic and employment-related attributes.

---

## 📊 Dataset Information

* Source: UCI Census Income Dataset
* Features include: Age, Workclass, Education, Occupation, Hours-per-week, etc.
* Target Variable: Income (<=50K / >50K)

---

## 🛠️ Tech Stack

* Python
* Pandas, NumPy
* Matplotlib, Seaborn
* Sklearn-

---

## 🔍 Approach

### 1. Exploratory Data Analysis (EDA)

* Distribution analysis
* Key insights extraction

### 2. Data Cleaning 

* Handled missing values
* Removed Null values
* Removed Duplicates

### 3. Model Building

* Applied classification algorithms:

  * Logistic Regression
  * Decision Tree
  * Random Forest

### 4. Model Evaluation

* Accuracy Score
* Classification Report

---

## 📈 Results

* Achieved accuracy: 84.40610484406105%
* Random Forest performed best among all models

---

## 💡 Key Insights

* Education and occupation significantly impact income
* Working hours strongly correlate with higher income
* Certain demographics show clear income trends
